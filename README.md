## 🎯 ميزات JavaScript

### 1. فئة ScrollAnimator
تحريك العناصر عند ظهورها في الشاشة:
```javascript
new ScrollAnimator();
```

### 2. فئة CounterAnimator
تحريك الأرقام بتأثير عد:
```javascript
new CounterAnimator();
```

### 3. فئة Modal
إدارة النوافذ المنفثقة:
```javascript
const modal = new Modal('modalId');
modal.open();
modal.close();
```

### 4. فئة FormHandler
معالجة وتحقق من النماذج:
```javascript
new FormHandler('form[data-contact-form]');
```

### 5. فئة MobileMenu
قائمة محمول قابلة للتبديل:
```javascript
new MobileMenu();
```

### 6. فئة BackToTop
زر العودة إلى الأعلى:
```javascript
new BackToTop();
```
